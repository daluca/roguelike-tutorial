# Rougelike tutorial

Following along with this [tutorial](https://bfnightly.bracketproductions.com/)

## Prerequisites

### rustup

```shell
rustup target add wasm32-unknown-unknown
cargo install wasm-bindgen-cli
```

### Arch Linux

```shell
pacman -Sy rust rust-analyzer rust-wasm rust-bindgen
```

## Run in docker

```shell
make run
```
